Приложение «калькулятор ЧСС»
Актуальность:
Тысячи врачей 
Цель: создать приложение, которое упростит жизнь тысячам врачей, попутно научившись работать с GUI “kivy” для языка программирования python.
Основные задачи:
1.	Научиться разрабатывать приложения на GUI “kivy”
2.	Научиться компилировать приложения, написанные на этом GUI, под андроид
3.	Изучить теорию по теме
4.	Написать приложение
Приложение будет рассчитывать целевую частоту сердечных сокращений для пациентов разного возраста во время проведения нагрузочного тестирования для диагностики ишемической болезни сердца.

Формула рассчета целевого уровня ЧСС следующая:
ЧСС = (220-h)*0,85 
где ЧСС – целевая частота сердечных сокращений
h – возраст

Подсчет ЧСС по клеточкам ЭКГ для скорости 25 мм/с
ЧСС= 60/(n*0,04)
Для скорости 50 мм/с 
ЧСС= 60/(n*0,02)
ЧСС- частота сердечных сокращений, уд/мин.
n- количество клеточек на ЭКГ
